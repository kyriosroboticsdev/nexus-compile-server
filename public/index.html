Library Template

Robot Config
Define your robot's drivetrain, mechanisms, and gear train. Mesh names must match OBJ groups exactly.
Drivetrain
Motors
Row 1: id, mesh name  ·  Row 2: role, cartridge, ext ratio, spin axis, reversed
Mechanisms
Group motors by subsystem. Key/Rev key: single character to hold for forward/reverse.
Pistons
Gears
Teeth / Input teeth: outputRPM = motorRPM × (inputT ÷ ownT). Meshing gears reverse direction.
Sprockets / Chain
Same direction as driver. outputRPM = inputRPM × (driverT ÷ ownT). Link to motor id or another sprocket id.
